📊 Member 2: LTV Business Analysis
Objective
Understand Customer Lifetime Value.

Step 1: Define LTV
Simple LTV:
EstimatedLTV = MonthlyCharges × tenure
Example:
MonthlyCharges = 100
Tenure = 20 months
LTV = 2000

Step 2: Create Target
df["EstimatedLTV"] = (    df["MonthlyCharges"]    * df["tenure"])

Step 3: Analyze Segments
df.groupby("Contract")["EstimatedLTV"].mean()
Insight:
Longer contracts generally produce higher LTV.

🌐 Member 4: FastAPI Development
Objective
Create prediction APIs.

API Structure
api/│├── main.py├── schemas.py├── database.py└── predict.py

schemas.py
from pydantic import BaseModelclass CustomerData(BaseModel):    tenure:int    MonthlyCharges:float

main.py
from fastapi import FastAPIimport joblibimport numpy as npfrom schemas import CustomerDataapp = FastAPI()churn_model = joblib.load(    "../models/churn_model.pkl")ltv_model = joblib.load(    "../models/ltv_model.pkl")

Churn API
@app.post("/predict_churn")def predict_churn(data: CustomerData):    features = np.array([        [data.tenure,         data.MonthlyCharges]    ])    pred = churn_model.predict(features)    return {        "ChurnPrediction":        int(pred[0])    }

LTV API
@app.post("/predict_ltv")def predict_ltv(data: CustomerData):    features = np.array([        [data.tenure,         data.MonthlyCharges]    ])    pred = ltv_model.predict(features)    return {        "EstimatedLTV":        float(pred[0])    }
